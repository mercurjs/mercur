<Snippet file="canary-warning.mdx" />

Mercur 2.0 is a complete rethink of how marketplace platforms should work. Instead of installing monolithic plugins, you pick and install individual **blocks** — modules, workflows, API routes, and UI extensions — directly into your project. You own every line of code.
