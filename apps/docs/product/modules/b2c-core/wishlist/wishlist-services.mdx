# createWishlists

This section provides a reference to the `createWishlists` method. This belongs to the Wishlist Module.

## createWishlists(data, sharedContext?): Promise\<WishlistDTO[]>

This method creates wishlist items.

### Example

```ts
const wishlists = await wishlistModuleService.createWishlists([
  {
    reference: "product",
    reference_id: "prod_123",
    customer_id: "cus_123",
  },
])
```

### Parameters

<ParamField body="data" type="CreateWishlistDTO[]" required>
  The wishlist items to be created.

  <Expandable title="properties">
    <ParamField body="reference" type='"product"' required>
      The type of the wishlist reference. Currently only supports `product`.
    </ParamField>

    <ParamField body="reference_id" type="string" required>
      The ID of the referenced entity (e.g., product ID).
    </ParamField>

    <ParamField body="customer_id" type="string" required>
      The ID of the customer who owns this wishlist item.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sharedContext" type="Context">
  A context used to share resources, such as transaction manager, between the application and the module.
</ParamField>

### Returns

<ResponseField name="Promise" type="Promise<WishlistDTO[]>">
  The created wishlist items.
</ResponseField>

# listWishlists

This section provides a reference to the `listWishlists` method. This belongs to the Wishlist Module.

This method retrieves a paginated list of wishlist items based on optional filters and configuration.

## Example

To retrieve a customer's wishlist:

```ts
const wishlists = await wishlistModuleService.listWishlists({
  customer_id: "cus_123",
})
```

To retrieve wishlist items for specific products:

```ts
const wishlists = await wishlistModuleService.listWishlists({
  reference_id: ["prod_123", "prod_456"],
})
```

By default, only the first `15` records are retrieved. You can control pagination by specifying the `skip` and `take` properties of the `config` parameter:

```ts
const wishlists = await wishlistModuleService.listWishlists(
  {
    customer_id: "cus_123",
  },
  {
    take: 20,
    skip: 2,
  }
)
```

## Parameters

<ParamField body="filters" type="FilterableWishlistProps">
  The filters to apply on the retrieved wishlist items.

  <Expandable title="properties">
    <ParamField body="id" type="string | string[]">
      The IDs to filter the wishlist items by.
    </ParamField>

    <ParamField body="customer_id" type="string | string[]">
      Filter wishlist items by their customer IDs.
    </ParamField>

    <ParamField body="reference" type="string | string[]">
      Filter wishlist items by their reference type.
    </ParamField>

    <ParamField body="reference_id" type="string | string[]">
      Filter wishlist items by their reference IDs (e.g., product IDs).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="config" type="FindConfig<WishlistDTO>">

    <ParamField body="skip" type="number">

    <ParamField body="take" type="number">

    <ParamField body="relations" type="string[]">
