# Mercur CLI

Add workflows, API endpoints, UI from registries to your project

## Installation

Install globally (recommended):

```bash
npm add -g @mercurjs/cli@latest
```

Any package manager works (npm, yarn, bun):

```bash
npm install -g @mercurjs/cli@latest
yarn global add @mercurjs/cli@latest
bun add -g @mercurjs/cli@latest
```

Installing it locally is also possible:

```bash
pnpm add -D @mercurjs/cli@latest
```
